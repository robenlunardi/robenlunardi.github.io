Pentest

Projeto de Pentest do Roben Lunardi :)

Projeto para invadirmos coisas e testar a segurança das coisas

Atualmente, as aplicações e serviços online oferecidos pelas instituições de ensino são amplamente utilizados. móveis são usados para lidar com diferentes tipos de dados. Normalmente, estes sistemas armazenam conjunto de informações pessoais que são armazenados e fornecidos para os usuários. Porém, muitas dessas aplicações são mantidas sem a avaliação de segurança necessária ou até mesmo mantidos de forma precária. A partir disso, os invasores podem tentar explorar os sistemas para obter ou causar vazamento de dados privados. Portanto, realizar avaliações de segurança é uma prática importante para encontrar vulnerabilidades nos serviços. Dentre as principais técnicas para avaliação de segurança, o Teste de Intrusão (Pentest) pode ser usado para detectar vulnerabilidades por meio de ataques simulados (ou até mesmo em ambientes de produção). Além disso, o Pentest pode ser realizado usando diferentes metodologias e melhores práticas, por meio de diversos frameworks para: organizar a execução do teste, executar ferramentas, fornecer estimativas, fornecer relatórios e documentar um Pentest. Desta forma, o objetivo principal deste projeto é avaliar diferentes modelos e metodologias de Pentest, utilizando como base sistemas educacionais. Como resultado, esperamos avaliar diferentes modelos de teste com estudo de caso em sistemas de instituições de ensino (sempre considerando o aval institucional para a realização dos testes em casos de sistemas não simulados).
